=====================================
PLANBOOKAI - DATABASE SETUP GUIDE
=====================================

QUICK START:
1. Khoi dong database:    .\scripts\start-dev-db.ps1
2. Dung database:         .\scripts\stop-dev-db.ps1

YEU CAU:
- Docker Desktop phai da cai dat va chay
- PowerShell

KET NOI DATABASE:
- Host: localhost
- Port: 5432
- Database: planbookai
- Username: test
- Password: test123
- JDBC: jdbc:postgresql://localhost:5432/planbookai

SCHEMAS (DDD):
- users: User Management (users, roles, sessions)
- content: Educational Content (lesson_plans, lesson_templates)
- assessment: Assessment (questions, exams, question_choices, exam_questions)
- students: Student Data (students, classes, student_results)

LENH HUU ICH:
- Xem logs:        docker logs planbookai-postgres-dev
- Restart DB:      docker restart planbookai-postgres-dev
- Xoa het data:    docker-compose -f docker-compose.dev.yml down -v

LOI THUONG GAP:
- Docker khong chay -> Mo Docker Desktop
- Port 5432 bi chiem -> Dung service khac dang dung port
- Script loi Unicode -> Dung PowerShell (khong phai Command Prompt)

SAMPLE DATA:
- 4 roles: ADMIN, MANAGER, STAFF, TEACHER
- 2 users: admin@planbookai.dev, teacher@planbookai.dev
- Password mac dinh: admin123 (BCrypt hash)

CHI TIET:
